# car-fuel-consumption-prediction

## Project Overview
**Goal:** Predict fuel consumption based on vehicle features to optimize fleet management.
**Author:** Marek Hoyler
**Date:** June 2025

## Description
This project uses a Random Forest Regressor to analyze car features (like horsepower, weight, height) and predict fuel efficiency.

## Results
- **R2 Score:** 0.9932
- **RMSE:** 0.2837
- **Top Feature:** Car Height & Horsepower

## How to run
1. Install requirements: `pip install -r requirements.txt`
2. Run the Jupyter Notebook.
